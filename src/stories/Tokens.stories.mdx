# Design Tokens

Tokens are the single source of truth for all design decisions in the Rabelo Digital Design System.
Import them as TypeScript constants or use them as CSS custom properties via `tokens.css`.

## Spacing

Based on a 4px base unit. Use `--ds-space-{n}` CSS variables.

| Token | Variable | Value |
|-------|----------|-------|
| 1 | `--ds-space-1` | 4px |
| 2 | `--ds-space-2` | 8px |
| 3 | `--ds-space-3` | 12px |
| 4 | `--ds-space-4` | 16px |
| 6 | `--ds-space-6` | 24px |
| 8 | `--ds-space-8` | 32px |
| 12 | `--ds-space-12` | 48px |
| 16 | `--ds-space-16` | 64px |

## Usage

```tsx
// TypeScript constants
import { semanticColors, spacing, typography } from "@rabelorp/ds-rd/tokens";

// CSS custom properties (add to your root CSS)
import "@rabelorp/ds-rd/tokens.css";
```
